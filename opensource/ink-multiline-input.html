Key Bindings

You can customize which keys trigger specific actions:

ControlledMultilineInput

For advanced use cases where you want to control the cursor position yourself, use ControlledMultilineInput:

import { ControlledMultilineInput } from "ink-multiline-input"; <ControlledMultilineInput value={text} cursorIndex={cursorPos} maxRows={10} showCursor={true} focus={true} />;
